A button pressing game, or i shall say, a striped down version of Simon Say with 3 modes
main.cpp contains the code for the ESP32
diagram.json for the diagram of the circuit
wokwi.toml to configure in VSC

The game has 3 modes, can be changed by pressing the mode button, pressing the mode button twice will trigger mode-indicator mode where all the light will shine, if mode 1 then blink once, others are the same


-mode 1: light - press button - repeat


-mode 2: press button - light - repeat


-mode 3: led blinking - press - repeat
